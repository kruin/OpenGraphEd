Edit Mode

To change the currently active Graph Editor Window into edit mode click the Edit Button, or choose Edit from the Modes Menu.

Actions within the Graph Editor Window

In Edit Mode, several commands can be performed by using the mouse and/or keyboard within the currently active Graph Editor Window:
CommandTriggerResulting Action
Node creation Single / Double click in empty space A new node is created at the current mouse location. (Note: See Preferences)
Node selection Single click on an existing node, or drag the mouse from an empty location to create a rectangle around the desired nodes The selection of the node is turned on (black dashed circle around) or off (Note: you can also select edges by dragging the mouse from an empty location, all nodes or edges within the green rectangle will be selected when the mouse button is released)
Node movement Press and drag the mouse over a selected node The selected node is dragged to a new location (Note: Holding down the Control key causes all selected nodes to be moved)
Node deletion Press the Delete Key All selected Nodes (and Edges) are deleted (Note: This is the same as the 'Remove Selected' menu / toolbar option)
Node labeling Right-click a node and click the label button from the popup. The node label is changed to the value you enter. (Note: Change to 'Show Labels' from the toolbar or display menu to see node labels) (Note: Holding down the Control key while right clicking allows the label of all selected nodes to be changed)
Node colouring Right-click a node and click the colour button from the popup. The node colour is changed to the value you enter. (Note: Holding down the Control key while right clicking allows the colour of all selected nodes to be changed)
Edge creation Press and drag the mouse over a non-selected node An edge will be created linking the non-selected node to whichever node the mouse is released over, or an edge will be created linking the non-selected node to a new node at the position where the mouse is released (Note: See Preferences)
Edge and Node creation Press and hold the control key and then press and drag the mouse from a non-selected node, or if enabled, press and drag the mouse from a non-selected node. An edge will be created linking the non-selected node to a new node at the position where the mouse is released. (Note: See Preferences)
Edge selection Single click on the 'bump' located at the center of the desired edge, or drag the mouse from an empty location to create a rectangle around the desired edges (centers points) The selection of the edge is turned on (black dashed line underneath) or off (Note: you can also select edges by dragging the mouse from an empty location, all nodes or edges within the green rectangle will be selected when the mouse button is released)
Edge deletion Press the Delete Key All selected Edges (and Nodes) are deleted (Note: This is the same as the 'Remove Selected' menu / toolbar option)
Edge order display Press and hold the control key and then click on a node or the center 'bump' of an edge The edges around the chosen node are flashed in order, or the edges bordering the same faces as the chosen edge are flashed in order (Note: Pressing and holding Control and Shift reverses the order for nodes)
Edge curving Select the edge to curve and drag its center point. Dragging the center point will make the edge be a curve going through that point as defined by a bezier curve. (Note: After curving an edge, the angle of the curve will be preserved when end points are moved)
Edge straightening Right-click an edge (center-point) and click the make straight button from the popup. The edge will become straight (its center-point will become aligned with the end points) (Note: Holding down the Control key while right clicking allows all selected edges to be straightened)
Edge directing Press the mouse over the source end Node and drag the mouse to the destination end Node. The Edge is directed (drawn with an arrow at the center).towards the destination Node. (Note: Directing an edge in the opposite direction of its existing direction will make the edge undirected)
Edge undirecting Press the mouse over the destination end Node and drag the mouse to the target end Node or, Right-click an edge (center-point or arrow) and click the Remove Direction button from the popup. The direction of the Edge (drawn with an arrow at the center).is removed. (Note: Holding down the Control key while right clicking allows all selected edges to be undirected)
Edge colouring Right-click an edge (center-point) and click the colour button from the popup. The edge colour is changed to the value you enter. (Note: Holding down the Control key while right clicking allows the colour of all selected edges to be changed)
