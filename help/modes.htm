Modes of Operation

JGraphEd allows graphs to be edited using several different modes of operation. These modes determine what action will be performed by certain mouse actions inside the current graph editor window.

The mode of a graph editor window can be changed by selecting the appropriate mode from the Mode menu or Mode button chooser on the toolbar.

The following modes of operation are available:
Edit Mode allows Nodes and Edges to be added, removed, moved, directed, curved or colored.
Move Mode allows the entire graph to be repositioned.
Rotate Mode allows the entire graph to be rotated around its center of gravity.
Resize Mode allows the entire graph to be resized or scaled.


Note: Each graph editor window will keep track of the mode it was in when it had the focus.
