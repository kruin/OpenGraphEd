# JGraphEd

Legacy Java graph editor application.

## Current state
Imported from a local 2004 source tree.

## Structure
- `dataStructure/`
- `graphException/`
- `graphStructure/`
- `operation/`
- `userInterface/`
- `images/`
- `help/`
- `JGraphEdFrame.java`
- `JGraphEdApplet.java`

## Notes
This is a legacy desktop/app/applet era Java project.
Modern browsers do not support Java applets, so the desktop entry point is the practical target.

## First goal
Get the desktop application compiling and starting locally.
